# SistemaLogistica


Proyecto en **Java 17** que simula un sistema básico de logística para gestionar transportes y calcular el consumo de combustible durante viajes.

##  Ejecución

Desde VS Code: ejecutar la clase que contiene:

```java
public static void main(String[] args)
```

Desde consola:

```
javac src/main/java/com/example/Main.java
java com.example.Main
```

## Objetivo

Aplicar conceptos de Programación Orientada a Objetos (POO) como herencia, métodos y uso de listas.
